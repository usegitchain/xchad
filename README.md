# xChad

AI moderator for X group chats. Scam links, impersonators, FUD floods — gone in milliseconds.

> Chad keeps X clean.

## Stack

- Next.js 15 (App Router)
- React 19
- Tailwind CSS
- TypeScript

## Develop

```bash
npm install
npm run dev
```

Open [http://localhost:4400](http://localhost:4400).

## Build

```bash
npm run build
npm start
```

## Notes

Not affiliated with X Corp. xChad is a third-party moderation layer built on the public X API.
